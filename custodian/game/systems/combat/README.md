# Combat Systems

- Belongs here: shared combat profiles, hit resolution helpers, combat resource/readability services, non-actor-specific combat simulation.
- Does not belong here: actor-local animation playback, HUD widgets, procgen construction.
- Current migration status: active folder; future work should prefer this domain over `game/systems/core/systems`.
- Current source of truth: `game/systems/combat/melee_attack_profile.gd` plus actor weapon definitions and current enemy/operator combat hooks.

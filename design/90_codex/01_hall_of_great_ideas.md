# Hall of Great Ideas

This is a short list of high-signal ideas worth revisiting. It is not a promise to implement them.

## Current Highlights

- Developer Observatory - shared debug visibility for invisible simulation state.
- World State Graph - reactive infrastructure dependencies across repairs, power, doors, and routes.
- Persistent World History - locations remember meaningful events.
- World Autopsy - terminals reconstruct local history from stored simulation events.
- Encounter Language - architecture teaches danger and opportunity without tutorials.
- Mystery Budget - each region deliberately preserves unanswered questions.
- Landmark Hierarchy - sectors orient players through memorable world structure.
- Navigation and Combat Heatmaps - playtest telemetry for pathing, danger, deaths, and combat flow.
